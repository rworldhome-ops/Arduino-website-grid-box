Arduino Uno

The Arduino Uno is a great choice for beginners and simple electronics projects. It allows you to read inputs from sensors and control outputs like LEDs, motors, and displays. Because it is easy to program and widely supported, it is often the first board people use when learning Arduino.

Key specifications

  • microcontroller: ATmega328P
  • operating voltage: 5V
  • digital I/O pins: 14 (6 support PWM)
  • analog input pins: 6
  • clock speed: 16 MHz
  • flash memory: 32 KB
  • SRAM: 2 KB
  • EEPROM: 1 KB
  • USB connection: USB Type-B
